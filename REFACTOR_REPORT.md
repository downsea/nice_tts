## 🚀 使用指南

### 安装和配置
```bash
cd nice_tts
pip install -e .
nice-tts config init  # 创建配置文件
# 编辑 .env 文件添加 API 密钥
```

### 基本使用
```bash
# 处理单个文件
nice-tts process meeting.wav

# 批量处理
nice-tts process audio_files/ --parallel 2

# 系统检查
nice-tts check-gpu
nice-tts list-models
```

### 运行测试
```bash
python simple_validation.py  # 核心功能验证
pytest tests/                # 完整测试套件
```
